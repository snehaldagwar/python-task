Defination of css

CSS stands for Cascading Style Sheets. It is a rule-based stylesheet language used to control the visual presentation, formatting, and layout of a web page written in a markup language like HTML or XML.While HTML establishes the structural skeleton of a web document, CSS provides the design, determining features like colors, typography, spacing, positioning, and responsive layouts for varying screen sizes. It is considered one of the three core cornerstone technologies of the World Wide Web, alongside HTML and JavaScript.
